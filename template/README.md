# Workshop Template

Use this template to create new Slidev workshops with the neobrutalist design system.

## Quick Start

```bash
./init.sh "Mi Nuevo Workshop" mi-nuevo-workshop
cd mi-nuevo-workshop
npm install
npm run dev
```

## What's Included

- `slides.md.template` - Starter slides with all custom layouts demonstrated
- `CLAUDE.md.template` - Claude Code instructions with design system rules
- `init.sh` - Bootstrap script that scaffolds a new repo

## Customization

1. Edit colors in `uno.config.ts`
2. Modify layouts in `layouts/`
3. Add components in `components/`
4. Update fonts in `styles/index.css`
